Como aplicar a otimização

O script cria a pasta C:\Collect automaticamente. Não precisa criar nada manualmente. Coloque o Collect_Free.ps1 e o Executar_Collect_Free.bat na mesma pasta e execute o .bat.
1
Clique duas vezes em Executar_Collect_Free.bat
O lançador pede permissão de administrador automaticamente — clique em Sim. Não precisa abrir o PowerShell manualmente nem configurar nada. Os dois arquivos (.bat e Collect_Free.ps1) precisam estar na mesma pasta.
2
Aguarde os downloads automáticos (2–5 min)
O script baixa todos os apps do pacote automaticamente via Winget e GitHub. Não feche a janela. Você verá cada arquivo sendo baixado na tela.
3
Ponto de Restauração — digite S e Enter
O script cria um ponto de restauração antes de qualquer mudança. Recomendamos aceitar — se algo der errado você pode voltar ao estado anterior pelo Painel de Controle → Restauração do Sistema.
4
No menu — pressione A para Aplicar Tudo
O menu tem 16 categorias. Pressione A para aplicar todas de uma vez. Se quiser aplicar só uma categoria específica, digite o número correspondente.

Aplicativos do pacote

CPU-Z

CZ
CPU-Z
cpu-z_2.19-en.exe
Monitor
Exibe informações detalhadas do hardware — processador, memória RAM, placa mãe e placa de vídeo. Essencial para saber exatamente o que tem dentro do PC antes de otimizar.
1
Abra o cpu-z_2.19-en.exe em C:\Collect
2
Aba CPU — veja o nome do processador, número de cores, velocidade e cache
3
Aba Memory — veja o tipo de RAM (DDR4/DDR5), frequência real e dual/single channel
Se a frequência mostrar 1333MHz ou 2133MHz mas a RAM é DDR4-3200, o XMP não está ativado no BIOS.
4
Aba Mainboard — veja fabricante e modelo da placa mãe
5
Aba Graphics — informações básicas da GPU
Quando usar
Antes de qualquer otimização para confirmar o hardware. Se precisar de suporte técnico, sempre informe o que aparece nas abas CPU e Memory.

HWiNFO64

HW
HWiNFO64
hwi64_846.exe
Monitor
Monitoramento em tempo real de temperatura, clock, tensão e uso de CPU, GPU e RAM. O mais completo para acompanhar o PC durante o jogo e verificar se há throttling térmico.
1
Abra o hwi64_846.exe e clique em Run
2
Selecione "Sensors only" na janela inicial
Isso abre só a lista de sensores — mais simples e direto.
3
Procure na lista: CPU Package (temperatura geral do CPU) e GPU Temperature
4
Procure por CPU Frequency — se durante o jogo a frequência cair abaixo do boost máximo, está em thermal throttle
Ex: i5-10400F tem boost de 4.3GHz. Se durante o jogo aparecer 2.9GHz, está throttlando por temperatura.
5
Para usar como overlay no jogo: instale o RTSS (RivaTuner) junto e ative o HWiNFO Shared Memory Support em Settings
Quando usar
Após a otimização, abra o HWiNFO e jogue por 30 minutos. Verifique se a temperatura do CPU ficou abaixo de 90°C e se a frequência manteve o boost. Se throttlar, considere melhorar a refrigeração ou usar o WeakPC Pro.
Overlay dentro do jogo
Para ver FPS e temperatura sem minimizar o jogo, instale o RivaTuner Statistics Server (RTSS) separado — disponível gratuitamente no site do MSI Afterburner.
1
No HWiNFO → Settings → aba General → marque "Shared Memory Support"
2
Abra o RTSS → Add → selecione o .exe do jogo
3
No HWiNFO → botão direito em qualquer sensor → "Add to OSD"
Recomendados: CPU Package Temp, CPU Frequency, GPU Temperature, GPU Core Clock, Framerate.

MSI Utility v3

MSI
MSI Utility v3
MSI_util_v3.exe
Otimização
Ativa o MSI Mode (Message Signaled Interrupts) na GPU e outros dispositivos. O MSI Mode cria um canal dedicado de comunicação entre a GPU e o CPU, reduzindo latência de input e melhorando o 1% low.
O script já tenta ativar o MSI Mode automaticamente via registry. Use o MSI Util para confirmar se está ativo e para reativar manualmente se necessário (ex: após atualizar driver).
1
Abra o MSI_util_v3.exe como Administrador
2
Localize sua GPU na lista (ex: NVIDIA GeForce RTX 3060)
3
Coluna MSI deve estar marcada — se não estiver, marque e clique em Apply
Se aparecer "Line-based" na coluna Mode, o MSI não está ativo. Marque e aplique.
4
Reinicie o PC após aplicar
Quando usar
Após instalar ou atualizar o driver da GPU. Atualizações de driver podem desativar o MSI Mode — verificar toda vez que trocar o driver.

ISLC — Intelligent Standby List Cleaner

ISL
ISLC
ISLC v1.0.4.5.exe
Otimização
Duas funções: mantém o Timer Resolution em 0.5ms (o padrão do Windows é 15.6ms) e limpa periodicamente a lista de standby da RAM — memória que o Windows reserva "por precaução" mas que o jogo poderia estar usando.
1
Abra o ISLC v1.0.4.5.exe
2
Marque "Enable custom timer resolution"
3
Defina o valor para 0.50ms
O campo "Current timer resolution" deve mostrar um valor próximo de 0.5ms após aplicar.
4
Marque "Start cleaning automatically" e defina quando limpar em 1024MB ou quando a RAM disponível cair abaixo de 25%
5
Clique em Start — minimize para a bandeja e deixe rodando em background antes de abrir o jogo
Quando usar
Abrir o ISLC antes de jogar sempre que possível, especialmente em PCs com 8GB de RAM. Com 16GB+ o benefício de limpeza de standby é menor, mas o Timer Resolution ainda ajuda.

DDU — Display Driver Uninstaller

DDU
Display Driver Uninstaller
DDU v18.1.5.2.exe
Driver
Remove completamente o driver de GPU (NVIDIA ou AMD) incluindo arquivos residuais que o desinstalador padrão deixa. Usado antes de instalar um driver novo limpo — evita conflitos, crashes e artefatos visuais causados por arquivos de driver antigo.
Usar sempre em Modo de Segurança. O DDU foi projetado para rodar sem o driver ativo — no modo normal o Windows pode reinstalar o driver automaticamente durante o processo.
1
Baixe o novo driver da GPU antes de começar (site da NVIDIA ou AMD)
2
Pause o Windows Update antes de continuar — Configurações → Windows Update → Pausar por 1 semana
Se não pausar, o Windows pode reinstalar o driver antigo automaticamente enquanto o DDU está removendo.
3
Reinicie em Modo de Segurança: Configurações → Sistema → Recuperação → Inicialização Avançada → Solucionar problemas → Opções avançadas → Configurações de inicialização → Reiniciar → F4
4
No Modo de Segurança, abra o DDU v18.1.5.2.exe
5
Selecione o tipo de GPU (GPU) e fabricante (NVIDIA ou AMD)
6
Clique em "Clean and restart"
O PC reinicia com o driver removido completamente.
7
Após reiniciar, instale o novo driver normalmente
Quando usar
Ao trocar de GPU, ao atualizar para uma versão major de driver, ou quando o jogo estiver crashando com erros de driver. Não é necessário em toda atualização de driver.

NVCleanstall

NVC
NVCleanstall
NVCleanstall_1.19.0.exe
Driver
Instala o driver NVIDIA removendo componentes desnecessários — GeForce Experience, telemetria, ShadowPlay, audio HDMI virtual e outros. Driver mais limpo = menos processos em background, menos RAM usada, menos interferência.
1
Abra o NVCleanstall_1.19.0.exe
2
Selecione "Download the recommended driver" — ele busca o driver mais recente automaticamente
3
Na tela de componentes, mantenha marcado apenas:
• Display Driver
• HD Audio via HDMI (se usar HDMI para som)
• PhysX (para jogos que usam)
Desmarque: GeForce Experience, Telemetry, Ansel, Highlights, NV Container, USB-C Driver (se não tiver USB-C na GPU)
4
Em Tweaks, ative: Disable MPO (evita tearing em dual monitor) e Use MSI after install
5
Clique em Install e aguarde o reinício
6
Reaplique o perfil NVIDIA: abra o nvidiaProfileInspector.exe em C:\Collect vidiaProfileInspector\ → File → Import → selecione Collect_Nvidia_Free.nip
Instalar um novo driver apaga o perfil NPI. Sempre reaplique após qualquer atualização de driver.
7
Verifique o MSI Mode: abra o MSI_util_v3.exe e confirme que a GPU está em modo MSI
Quando usar
Ao instalar ou atualizar o driver NVIDIA. Usar sempre junto com o DDU para resultado mais limpo.

GeekUninstaller
geek.exe
Utilitário
Desinstalador mais completo que o padrão do Windows. Remove o app e todos os arquivos residuais — registry, pastas ocultas, entradas de inicialização. Essencial para limpar o PC antes de otimizar.

Abra o geek.exe — ele não precisa de instalação

Para remover qualquer programa que suspeitar estar pesando no PC — especialmente lixo como ClickShare, McAfee, Candy Crush e apps que vieram pré-instalados. Sempre preferir o GeekUninstaller ao "Programas e Recursos" do Windows.

Autoruns

AUTO
Autoruns (Sysinternals)
Autoruns.exe
Utilitário
Ferramenta da Microsoft (Sysinternals) que lista tudo que inicia automaticamente com o Windows — muito mais completo que o Gerenciador de Tarefas. Encontra malware, bloatware e programas que consomem recursos sem você saber.
Cuidado ao desativar itens. Entradas de sistema (Microsoft, Windows) devem ser mantidas. Desativar o item errado pode causar instabilidade. Em caso de dúvida, pesquise o nome no Google antes de desativar.
1
Abra o Autoruns.exe como Administrador
2
Vá em Options → Filter Options → marque "Hide Microsoft entries" e "Hide Windows entries"
Isso esconde as entradas seguras e mostra só terceiros — muito mais fácil de analisar.
3
Aba Logon — programas que iniciam com o Windows. Desmarque o que não precisa (Discord, Spotify, Steam, OneDrive, etc.)
4
Aba Scheduled Tasks — tarefas agendadas de terceiros. Entradas com Publisher vazio e caminho suspeito merecem atenção.
5
Entradas destacadas em amarelo = arquivo não encontrado. Podem ser removidas com segurança.
Quando usar
Se o PC estiver lento ao iniciar ou se suspeitar de malware ou programas indesejados rodando em background. Use uma vez por mês para manter o startup limpo.

Configurações de Firmware

BIOS — Configurações Recomendadas

O nome das opções varia por fabricante (ASUS, Gigabyte, MSI, ASRock). Procure pelo conceito — não pelo nome exato. A maioria está em Advanced → CPU Configuration ou AI Tweaker.
XMP / DOCP / EXPO — o mais importante
ENABLED
Ativa o perfil de velocidade real da RAM. Sem XMP, DDR4-3200 roda a 2133MHz — 30% mais lento do que deveria. É o erro mais comum em PCs sem que o usuário saiba.

Como verificar: Abra o CPU-Z → aba Memory → campo "DRAM Frequency". Multiplique por 2 — se o resultado for menor que a frequência da sua RAM, o XMP não está ativo.

Onde ativar: BIOS → AI Tweaker / OC / DRAM Configuration → XMP = Profile 1.
Intel SpeedStep / AMD Cool'n'Quiet
DISABLED
Reduz o clock do CPU em baixa carga. Causa ramp-up lento quando o jogo exige mais de repente — stutter nos primeiros frames de um confronto.

Onde desativar: BIOS → Advanced → CPU Configuration → Intel SpeedStep (ou Power Management → Cool'n'Quiet no AMD).
Turbo Boost / Precision Boost
ENABLED
Alguns fabricantes desativam por padrão em placas entry-level. Sem boost, o CPU roda sempre no clock base — pode ser 1GHz abaixo do boost máximo.

Onde verificar: BIOS → CPU Configuration → Intel Turbo Boost / AMD Precision Boost → Enabled.
Fast Boot / Ultra Fast Boot
DISABLED
Pode causar instabilidade após atualizações de driver ou Windows. O ganho de 2-3 segundos no boot não justifica o risco.
Above 4G Decoding + Resizable BAR
ENABLED
Permite ao CPU acessar toda a VRAM da GPU. Em RTX 3000+ e RX 6000+ com placa mãe compatível (Z490+, B550+), traz 5-15% de FPS em jogos compatíveis. Ativar os dois juntos.

Onde ativar: BIOS → Advanced → PCI Subsystem Settings → Above 4G Decoding = Enabled + Re-Size BAR Support = Enabled.
Configure uma vez, não precisa mais voltar. As configurações de BIOS são permanentes — ficam salvas mesmo após formatar o Windows. Após configurar, o script cuida do resto.
Performance Extra — Opcional

RAM — Overclock seguro

Opcional mas recomendado. XMP já é o passo mais importante. O que está abaixo vai além do XMP para quem quer extrair ainda mais sem riscos.
Timings primários — CL mais baixo
Seguro
Reduzir o CL (CAS Latency) melhora a latência de memória sem aumentar calor. Na mesma frequência, um CL menor significa que a RAM responde mais rápido.
DDR4-3200
CL14-16
DDR4-3600
CL16-18
DDR5-5600+
CL28-30
Se não souber a frequência real da RAM: CPU-Z → aba Memory → DRAM Frequency × 2 = frequência real.
Infinity Fabric FCLK — AMD Ryzen
Sweet spot
O Ryzen sincroniza RAM e CPU via Infinity Fabric em modo 1:1. O sweet spot é DDR4-3600 com FCLK 1800MHz — melhor latência e bandwidth juntos.

Como ativar: BIOS → AI Tweaker → DRAM Frequency = 3600 + FCLK Frequency = 1800MHz. Se a RAM não suportar 3600, use 3200 com FCLK 1600MHz.
PBO + Curve Optimizer — AMD Ryzen
Seguro
O PBO é o overclock oficial da AMD — deixa o CPU boostar além do padrão usando o próprio algoritmo da AMD. Não viola garantia. Com Curve Optimizer negativo, ganha-se +100-200MHz no boost e -5-10°C de temperatura.

Como ativar: BIOS → AMD Overclocking → Precision Boost Overdrive → Advanced → PPT 200W · TDC 140A · EDC 200A + Curve Optimizer All Core = -10. Testar com Cinebench R23 por 20 minutos. Se travar, ajustar para -5.
Power Limits PL1/PL2 — Intel não-K
Seguro
CPUs Intel sem sufixo K (i5-12400, i7-12700 etc.) têm frequência bloqueada, mas muitas placas limitam o power abaixo do máximo da Intel. Remover esse limite faz o CPU manter o boost por muito mais tempo.

Como ativar: BIOS → CPU Configuration → Power Limits → PL1 = 253W · PL2 = 253W · Tau = 56. Monitorar temperatura com HWiNFO — se ultrapassar 90°C, reduzir o PL1.
Requer bom cooler — tower ou AIO 240mm+. Com cooler stock Intel box não recomendado.
